# 3.基础语法

## 3.1 注释

~~~ python
# 第一个注释
# 第二个注释
 
'''
第三注释
第四注释
'''
 
"""
第五注释
第六注释
"""
print ("Hello, Python!")
~~~

## 3.2 多行语句

~~~ python
total = item_one + \
        item_two + \
        item_three
~~~

## 3.3 print 输出

~~~python
# 换行输出
print(x)
# 不换行输出
print(x, end="")
~~~

## 3.4 import 导入

~~~python
# 将整个模块(somemodule)导入
import somemodule
# 从某个模块中导入某个函数
from somemodule import somefunction
# 从某个模块中导入多个函数
from somemodule import firstfunc, secondfunc, thirdfunc
# 将某个模块中的全部函数导入
from somemodule import *
~~~
